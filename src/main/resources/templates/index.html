Simple factory authorization api car application
Sign in  Sign up Home

Application contains data about fifty cars and delivers eleven endpoints including one available to non-registered users, the rest require an account and confirmation via email. A registered user has the ability to precisely download data from api and use tools such as Swagger UI.

GET /api/cars
example for localhost: http://localhost:8080/api/cars

Endpoint allowed without registration:
  • api/cars/all - return all cars
Application has one existing static active user:

userEmail: user@user.com
password: user
